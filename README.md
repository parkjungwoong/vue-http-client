# vue-http-client
axios를 패키징하여 서비스 로직에서 좀더 사용하기 편하도록 만들어봤습니다.
특히 jwt 토큰을 이용하는 서비스에 대응할 수 있도록 만들어졌습니다.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
